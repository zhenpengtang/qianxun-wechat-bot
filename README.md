# 📱 微信群管理工具

> 基于千寻微信 API 的 Web 管理界面，支持群列表管理、批量消息发送等功能

## ✨ 功能特性

- 🔄 **自动获取群列表** - 一键更新并保存所有微信群
- 📤 **批量消息发送** - 支持向多个群同时发送消息
- ✅ **发送状态追踪** - 实时显示每个群的发送结果
- 📜 **发送历史记录** - 保留最近 50 条发送记录
- 🎨 **简洁界面** - Bootstrap 5 响应式设计，支持移动端

## 🚀 快速开始

### 环境要求

- PHP 7.4+
- cURL 扩展
- JSON 扩展

### 安装步骤

1. **克隆或下载项目**
   ```bash
   git clone <repository-url>
   cd wechat-group-manager
   ```

2. **配置文件**
   
   编辑 `sender.php`，修改 API 配置：
   ```php
   $API_HOST = "http://192.168.66.42:7777";
   $WXID = "wxid_q44zpd46iugs22";  // 你的微信 ID
   ```

3. **设置权限**
   ```bash
   chmod 644 sender.php
   chmod 777 groups_selected.json send_log.json  # 或确保目录可写
   ```

4. **启动 Web 服务器**
   ```bash
   # PHP 内置服务器
   php -S 0.0.0.0:8080
   ```
   
   或使用 Apache/Nginx 指向项目目录

5. **访问界面**
   ```
   http://localhost:8080/sender.php
   ```

## 📖 使用说明

### 1️⃣ 更新群列表

- 进入 **群列表** 标签页
- 点击 **🔄 更新并保存所有群**
- 系统会自动获取并保存所有微信群到 `groups_selected.json`

### 2️⃣ 发送消息

- 切换到 **发消息** 标签页
- 勾选要发送的群（支持全选/取消）
- 输入消息内容
- 点击 **🚀 发送消息**

### 3️⃣ 查看结果

- 发送完成后显示成功/失败统计
- 详细列表显示每个群的发送状态
- 底部显示历史发送记录

## 📁 文件说明

| 文件 | 说明 |
|------|------|
| `sender.php` | 主程序文件（整合界面 + API） |
| `groups_selected.json` | 保存的群列表数据 |
| `send_log.json` | 发送历史记录 |

## 🔌 API 接口

### 获取群列表

```json
POST /qianxun/httpapi?wxid={WXID}
{
  "type": "Q0006",
  "data": {"type": "1"}
}
```

### 发送消息

```json
POST /qianxun/httpapi?wxid={WXID}
{
  "type": "Q0001",
  "data": {
    "wxid": "xxx@chatroom",
    "msg": "消息内容"
  }
}
```

## ⚙️ 消息格式

支持以下格式：

- **普通文本**: `你好世界`
- **@提醒**: `[@,wxid=wxid_xxx,nick=昵称,isAuto=true]`
- **表情符号**: `[emoji=D83D][emoji=DE01]` 或 `[捂脸][奸笑]`

## 🛠️ 故障排除

### 无法获取群列表
- 检查 API 地址是否正确
- 确认微信 API 服务正常运行
- 检查网络连接

### 发送失败
- 确认账号登录状态正常
- 检查消息内容是否包含敏感词
- 降低发送频率

### 文件无法写入
```bash
chmod 777 groups_selected.json send_log.json
# 或
chown www-data:www-data *.json
```
